# BlazeMates

A cannabis enthusiast community platform for connecting people who share a passion for cannabis culture, cultivation, and community.

## What is BlazeMates?

BlazeMates is a social networking platform designed for the cannabis community. Connect with fellow enthusiasts, share cultivation tips, join group discussions, attend local events, and engage with like-minded people in a welcoming environment.

## Features

- **Community Feed**: Share posts, photos, and updates with the community
- **Group Chats**: Join interest-based groups for cultivation tips, strain discussions, and more
- **Local Events**: Discover and attend cannabis-related events in your area (smoke sessions, educational meetups, etc.)
- **Direct Messaging**: Connect and chat with other community members
- **Gamification**: Earn Blaze Levels by being active and engaged in the community
- **Premium Tiers**: Unlock additional features with Blaze+ and Blaze Pro subscriptions

## Platform Focus

BlazeMates is NOT a dating platform. It is a community-focused social network for cannabis enthusiasts to:
- Share cultivation knowledge and growing tips
- Discuss strains and consumption methods
- Organize and attend local community events
- Build friendships within the cannabis community
- Support cannabis advocacy and education

## Technology Stack

- React Native with Expo
- Supabase (Database & Authentication)
- TypeScript
- Expo Router for navigation

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Set up environment variables in `.env`

3. Start the development server:
```bash
npm start
```

4. Run on your preferred platform:
```bash
npm run web     # Web
npm run android # Android
npm run ios     # iOS
```

## Community Guidelines

BlazeMates is committed to providing a safe, respectful environment for all cannabis enthusiasts. Users must:
- Be 21+ years of age (or local legal age)
- Follow all local laws and regulations
- Treat all community members with respect
- Report inappropriate behavior
